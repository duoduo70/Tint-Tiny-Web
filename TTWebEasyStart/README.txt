First, you must see frpc.ini .
Sencond, you can direct run TTWebEasyStart.exe .
But if you want auto-start in computer start:
Create a shortcut for TTWebEasyStart.exe. Move it to:
%AppData%\Microsoft\Windows\Start Menu\Programs\Startup
And restart your computer.

中文：
首先你要配置 frpc.ini
然后，你可以直接运行 TTWebEasyStart.exe 。
如果你想在电脑启动时就自动启动软件：
为 TTWebEasyStart.exe 创建快捷方式。将快捷方式移动到： 
%AppData%\Microsoft\Windows\Start Menu\Programs\Startup 中。
然后重启电脑。
