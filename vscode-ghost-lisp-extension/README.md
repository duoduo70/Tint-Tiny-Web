# Ghost Lisp
Support for Ghost Lisp programming language (to expand Tiny Tiny Web).

F4: Start REPL
Shift+F4: Start REPL (debug mode)

F5: Test a file
Shift+F5: Test a file (debug mode)

(Repo: https://github.com/duoduo70/Tiny-Tiny-Web/)
